Coersion - Truthy & Falsy

Bài 1:
var a = 10;
var b = Number(a);
var c = a * 1;
console.log(a) => 10;
console.log(b) => 10;
console.log(c) => 10;
Ta thấy kiểu dữ liệu của a, b, c đều là number => coercion là chuyển đổi từ kiểu dữ liệu này sang kiểu dữ liệu khác.
Ở ví dụ trên b và c đang có kiểu dữ liệu undefined và chuyển sang kiểu dữ liệu number.

Bài 2:
Khi một giá trị không phải Boolean cần chuyển sang Boolean thì:
- Nếu giá trị đó là một trong các giá trị undefined, null, NaN, 0, "" (chuỗi rỗng) => return false
- Các giá trị còn lại => return true
var a = 10;
var b = '';
if (a) {
console.log('a is truthy');
}
a có kiểu dữ liệu là number, value = 10 => if (a) return true => log ra "a is truthy"
if (b) {
console.log('b is truthy');
}
b có kiểu dữ liệu là string, value rỗng => if (b) return false => skip lệnh console.log('b is truthy') => không log ra gì hết
